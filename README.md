# 🍽️ 点菜小助手

一个简单的网页版点菜系统，让你对象可以在手机上选菜。

## 功能特点

- ✅ 图文菜单，30道家常菜预设
- ✅ 分类浏览（荤菜/素菜/汤品/主食/凉菜）
- ✅ 多选菜品 + 每道菜可以加备注
- ✅ 历史记录保存（最近30天）
- ✅ 管理后台可添加新菜品
- ✅ 手机友好，微信里打开就能用

---

## 使用方法

### 方法一：本地运行（推荐）

1. **双击运行 `start.sh` 或在终端运行：**
   ```bash
   cd /Users/chengorange/.qclaw/workspace/food-menu
   python3 server.py
   ```

2. **终端会显示地址，比如：**
   ```
   📱 请在手机浏览器打开:
      http://192.168.1.100:8080
   ```

3. **把地址发给你对象，手机打开即可点菜**

4. **管理后台（添加新菜品）：**
   ```
   http://192.168.1.100:8080/admin.html
   ```

### 方法二：部署到公网（可选）

如果想让链接长期可用，可以部署到：
- Vercel（免费）
- Netlify（免费）
- GitHub Pages（免费）

## 目录结构

```
food-menu/
├── index.html      # 点菜页面（给你对象用）
├── admin.html      # 管理页面（添加菜品）
├── style.css       # 样式
├── app.js          # 点菜逻辑
├── dishes.js       # 菜品数据
├── server.py       # 本地服务器
├── start.sh        # 一键启动脚本
└── README.md       # 使用说明
```

---

## 注意事项

- 本地运行时，电脑和手机必须在同一 WiFi 下
- 数据存储在浏览器本地（localStorage），换浏览器/设备数据不共享
- 如需长期保存，建议部署到公网

---

## 自定义菜品

1. 打开管理后台：`http://[IP]:8080/admin.html`
2. 填写菜品名称、分类、描述、标签
3. 点击"添加菜品"
4. 新菜品会立即出现在点菜页面
